# CentOS7でマイクラ鯖を構築
- CentOS7でマインクラフトサーバーを構築するBashスクリプトです。
- 操作を間違えるとデータが損失したり、サーバーにSSHなどのターミナルで接続できなくなる可能性があります。自己責任で実行してください。

# 実行環境
- CentOS7 (CentOS-7-x86_64-Minimal-1804.isoを利用)
- CentOS7のセットアップ時に以下を必ず設定してください。
 - ホスト名
 - ネットワーク設定（）





# 作成手順
